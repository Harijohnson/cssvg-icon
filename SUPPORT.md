# Support

Thank you for using `cssvg-icons`! We want to ensure you have the best experience possible.

## 🆘 How to Get Help

### 1. Documentation

Please check the [README.md](README.md) for installation instructions, usage examples, and configuration options.

### 2. GitHub Issues

If you encounter a bug or have a feature request, please search the [existing issues](https://github.com/Harijohnson/cssvg-icon/issues) to see if it has already been reported. If not, feel free to [open a new issue](https://github.com/Harijohnson/cssvg-icon/issues/new).

When reporting a bug, please include:

- A clear description of the issue.
- Steps to reproduce the behavior.
- Screenshots (if applicable).
- Your environment (OS, Node version, Browser).

### 3. Discussions

For general questions, design tips, or showing off what you've built, please use the [GitHub Discussions](https://github.com/Harijohnson/cssvg-icon/discussions) (if enabled) or comment on our issue tracker.

## 🛠️ Feature Requests

We love hearing new ideas! If you have a suggestion for a new icon or a feature, please open an issue with the "feature-request" label.

## 🤝 Contributing

Interested in contributing? Check out our [Contributing Guidelines](CONTRIBUTING.md) (coming soon) or simply submit a Pull Request!
