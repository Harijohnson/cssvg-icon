# Getting Started

## Installation

```bash
# npm
npm i cssvg-icons

# pnpm
pnpm add cssvg-icons

# yarn
yarn add cssvg-icons

# bun
bun add cssvg-icons
```

## Local Development

1. **Install dependencies**:

   ```bash
   bun install
   ```

2. **Start the development server**:

   ```bash
   bun dev
   ```

   Open [http://localhost:3000](http://localhost:3000) to view the documentation.
